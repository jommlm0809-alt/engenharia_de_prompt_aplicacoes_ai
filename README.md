# engenharia_de_prompt_aplicacoes_ai
Atividades da faculdade


Este repositório reúne materiais, projetos e experimentos desenvolvidos na disciplina Engenharia de Prompt e Aplicações em Inteligência Artificial.
A proposta da disciplina explorar como interagir de forma eficiente com modelos de Artificial Intelligence, utilizando técnicas de Prompt Engineering para criar aplicações inteligentes baseadas em Natural Language Processing.
